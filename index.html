  1. Maak een genummerde lijst waarin je uitlegt wat de betekenis is van ieder van de volgende begrippen:
    1. Transmission Control Protocol (TCP):
      -Verantwoordelijk voor besturing en verificatie van gegevens.
    2. Internet Protocol (IP):
      -Verantwoordelijkvoor adressering van computer naar computer.
    3. omain Name Servers (DNS):
      -De naam van een website, waar deze opgezocht kan worden.
    4. Router:
      -Zorgen dat gegeven op de goede plek aan komen, zij kiezen de meest optimale route.
    5. Hyper Text Markup Language (HTML)
      -Taal voor het beschrijven van een pagina en indelen van tekst.
    6. HTML tag:
      -Een (HTML)tag is een semantische afbakening van informatie waar het een uitspraak over doet

  2. Hoe weet een browser wat voor soort bestand door de server wordt geretourneerd?
    Browser weet via content-type welk soort bestand wordt gestuurd

  3. Wat is het doel van keywords bij een website? Wie of wat gebruikt deze?
    Deze zijn bedoelt om de website te kunnen vinden en worden gebruikt door de search engine!

  4. Wat is het doel van een description? Waar zal deze, wanneer je je site publiceert, te zien zijn?
    Een description geeft een omschrijving van de website en is te zien in de search engine!

  5. Klik in je browser met je rechter muisknop op je pagina, en kies Pagina weergeven (of Ctrl+U). Wat zie je?
    Je ziet de source-code van de website verschijnen!

  6. Klik op dezelfde wijze, en selecteer "Element inspecteren" (of Ctrl+Shift+I). Beweeg met je muis over het scherm. Wat zie je? Wat is het nut hiervan?
    Verschillende dingen worden ge-highlight en je kunt onderin het scherm de source-code zien! Het is een snelle manier van items controleren op je website!

Week 2

  1. Waarom behouden de headers uit opdracht 16 de originele kleur?
    Omdat deze in de html code zijn geschreven en dus als het ware de css overschrijven.

  2. Hoe komt het dat de rest van de stijl wel toegepast wordt?
    Omdat deze in de css zijn veranderd en in-line geen style hebben gekregen.

  3. Welke manieren zijn er om de waarde van een kleur aan te geven?
    RGB, HEX, HSL, RGBA, HSLA en bij basis kleuren de engelse naam van de kleur.
